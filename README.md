## Building

Prerequisites:
- Go lang 1.21 or higher.
- GNU Make.

Steps:
- Clone this repo and open a terminal in the folder where you cloned it.
- Run `make`.

The generated binaries will be located at `cmd/bin/`. Choose the one for your operating system and architecture.
